## JavaScript darkmode counter

This is a beginner JavaScript project that demonstrates DOM manipulation, event handling and local storage.

## Features
 - Toggle between light and dark mode
 - Count the number of times theme is changed
 - Text in button gets updated upon change of theme
 - Local storage and persistence for theme and counter

## Technologies used
 - HTML
 - CSS
 - JavaScript
 - Local storage
